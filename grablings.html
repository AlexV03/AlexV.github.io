Project Duration: 48 hours
Role: Gameplay Programmer & Technical Designer
Team Size: 7
Engine: Unreal Engine 5

About

Grablings is a game jam game made in 48 hours. The game is about climbing as high as possible with your friend(s) by grabbing each other and placing or throwing your friend onto platforms.

Creations I Made

In this game jam, I primarily created the UI logic for the game and designed the different rooms. I had never created any UI logic in UE5 before, and it was an interesting experience to try out.

The UI works with both keyboard/mouse and controller. With the help of talented artists who created art assets quickly, I was able to rapidly build out the different rooms.

My Thoughts

I really enjoyed participating in this game jam and had a lot of fun brainstorming what kind of game we could make and then creating it. I had a talented and great team, and I think we made a fun little local co-op game in the time we had.
