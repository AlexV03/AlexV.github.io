Project Duration: 48 hours
Role: Gameplay programmer & Technical designer
Team Size: 7
Engine: Unreal Engine 5

About:

Grablings is a game jam game. We made the game in 48 hours.
The game is about getting as high as possible together with your friend(s) by grabbing
each other and placing or throwing your friend to a platform.


Creations I made:

In this game jam I mainly created the UI logic for the game and created the different rooms.
I had never created any UI logic in UE5 before and it was an interesting experience to try out.
The UI works with keyboard and mouse and with a controller.
I was able to create the different rooms very fast with the help of very talented artists.
They created the art assets very quickly, and with that I was able to quickly make the rooms.


My thoughts:

I really enjoyed participating in this game jam and had a lot of fun brainstorming
what kind of game we could make and then creating the game.
I had a talented and great team, and I think we made a fun little local co-op game in the time we had.
